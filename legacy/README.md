# Euterpe

Euterpe is a self-hosted blind-test server written in Python. You can supply Euterpe with song extracts, run the web server, and access it through any browser.

## Development State

Euterpe is currently under heavy development (let's say pre-pre-alpha stage) and is not functional at all.. for now ! :)
